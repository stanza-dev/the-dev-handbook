# Contributing

Thanks for helping improve these courses!

## What we accept

- **Typo fixes** — always welcome
- **Clarity improvements** — better explanations, updated examples
- **New code examples** — practical, real-world snippets
- **Resource links** — official docs, high-quality articles

## What we don't accept

- **New courses** — course structure is managed by the Stanza team
- **Challenge content** — challenges are part of the interactive platform
- **Marketing content** — keep it educational

## How to contribute

1. Fork this repository
2. Create a branch: `git checkout -b fix/typo-in-usestate`
3. Make your changes
4. Submit a Pull Request with a clear description

## Style guide

- Use TypeScript for all code examples (unless the course is for another language)
- Include type annotations where applicable
- Keep examples practical and concise
- Link to official documentation when referencing APIs
